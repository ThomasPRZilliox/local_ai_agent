# Local AI Agent

This repository gathers multiple AI agent that aims to run locally

## How to set it up?

The article [Build Your Own AI Agent Locally with Google ADK](https://medium.com/@thomas.zilliox/build-your-own-ai-agent-locally-with-google-adk-7159286f1954)
will get you through on how to install Ollama and set your environment.
## Dice Agent demo

Create a new agent

```commandline
adk create dice_agent
```

Run the agent (CLI)

```commandline
adk run dice_agent
```
Run the agent (web)

```commandline
adk web
```

To stop the chat:
```commandline
exit
```
